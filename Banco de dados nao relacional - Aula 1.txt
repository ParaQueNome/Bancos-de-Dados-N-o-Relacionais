Introdução à Banco de Dados não Relacionais


Relacional X Não Relacional


Dados = Valor atribuido a determinado item.
Ex: Numeros, conceitos, posições;

Podem ser mensurados
O dado é a base para a informação
Informação é o conteúdo entendível, capaz de expressar uma função.
Dado é um elemento bruto sem significado por si só. Pode ser quantitativo ou qualitativo.
Já a informação é o conjunto de dados agrupados, os quais são tratados, interpretados e analisados com o objetivo de fomentar a construção
de conhecimento sobre determinado assunto.
